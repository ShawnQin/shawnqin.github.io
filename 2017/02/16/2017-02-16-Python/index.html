Jupyter notebook

  • Installation for both python2 and python3 . This website provides instruction to install both jupyter for both versions.
  • Open Jupyter Notebook. First, type source activate py3 in the promte, and then typ jupyter notebook
  • Solve “doesn’t understand the open location “ problem This probably a Mac issue. One trick is by specifying the web surface in ~/.jupyter/jupyter_notebook_config.py by adding c.NotebookApp.browser = u'chrome'.
  • trouble shooting tensorflow
  • When I install anaconda in my mac pro, the jupyter notebook works well, but when I type import tesorflow as tf, it popout couldn’t find module. Here is how I fix this bug
  • Create an environment for tensorflow conda -n tensorflow
  • Activate this environment source activate tensorflow
  • Check what are the jupyter and ipython verson. They are probably not in this environment. To intall them, type conda install ipython and pip install jupyter (in my cas pip3 doesn’t work, although I used python 3). Now tpyte which ipython and which jupyter to check if they have been installed properly

  • To check what python packages are in the tensorflow virtual environemnt, conda list. Install other package by conda isntall packagename
